# CS2 Major 2024 — Top Performer Prediction 🎮

Machine learning project to predict top-performing players 
at the PGL CS2 Major Copenhagen 2024 tournament.

## 📊 Dataset
- **PGL CS2 Major Copenhagen 2024** (Kaggle)
- 80 players, 39 features (K/D, ADR, KAST, headshot %, score...)

## 🧠 Models Used
- Random Forest Classifier
- Gradient Boosting Classifier

## 📈 What It Does
- Analyzes player statistics from the Major
- Predicts whether a player is a "Top Performer" based on ADR
- Visualizes key features: kill count, K/D ratio, headshot %, KAST
- Identifies the most important stats that separate elite players
